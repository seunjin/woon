# @woon-ui/combobox

Combobox primitives for searchable option picking.

## Installation

```bash
pnpm add @woon-ui/combobox
```

## Usage

```tsx
import { Combobox } from '@woon-ui/combobox'
import '@woon-ui/combobox/css'
```
