# @woon-ui/drawer

Edge-attached surfaces built on the woon dialog system.

## Installation

```bash
pnpm add @woon-ui/drawer
```

## Usage

```tsx
import { Drawer } from '@woon-ui/drawer'
import '@woon-ui/drawer/css'
```
