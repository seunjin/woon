# @woon-ui/dialog

Dialog primitives, `alert()`, `confirm()`, and plugin registration for Woon.

## Installation

```bash
pnpm add @woon-ui/dialog
```

## Usage

```tsx
import { Dialog } from '@woon-ui/dialog'
import '@woon-ui/dialog/css'
```

## CSS entry points

- `@woon-ui/dialog/css`
- `@woon-ui/dialog/css/dialog`
- `@woon-ui/dialog/css/alert`
- `@woon-ui/dialog/css/confirm`
