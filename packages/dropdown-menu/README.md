# @woon-ui/dropdown-menu

Dropdown menu primitives for trigger-based action menus.

## Installation

```bash
pnpm add @woon-ui/dropdown-menu
```

## Usage

```tsx
import { DropdownMenu } from '@woon-ui/dropdown-menu'
import '@woon-ui/dropdown-menu/css'
```
