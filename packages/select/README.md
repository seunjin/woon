# @woon-ui/select

Select primitives for keyboard-accessible custom select UIs.

## Installation

```bash
pnpm add @woon-ui/select
```

## Usage

```tsx
import { Select } from '@woon-ui/select'
import '@woon-ui/select/css'
```
