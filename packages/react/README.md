# @woon-ui/react

Meta package for Woon. Re-exports `ModalRoot`, `Toaster`, and component packages from a single entry.

## Installation

```bash
pnpm add @woon-ui/react
```

## Usage

```tsx
import { ModalRoot } from '@woon-ui/react'
import '@woon-ui/react/css'
```
