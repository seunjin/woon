# @woon-ui/react

Meta package for Woon. Includes `WoonProvider`, `useDialog`, and component re-exports.

## Installation

```bash
pnpm add @woon-ui/react
```

## Usage

```tsx
import { WoonProvider } from '@woon-ui/react'
import '@woon-ui/react/css'
```
