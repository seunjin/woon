# @woon-ui/tooltip

Tooltip primitives for hover, focus, and keyboard-triggered help text.

## Installation

```bash
pnpm add @woon-ui/tooltip
```

## Usage

```tsx
import { Tooltip } from '@woon-ui/tooltip'
import '@woon-ui/tooltip/css'
```
