# @woon-ui/context-menu

Context menu primitives for right-click or long-press interaction.

## Installation

```bash
pnpm add @woon-ui/context-menu
```

## Usage

```tsx
import { ContextMenu } from '@woon-ui/context-menu'
import '@woon-ui/context-menu/css'
```
