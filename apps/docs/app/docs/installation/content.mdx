## CSS 가져오기

앱 진입점에 CSS를 한 번 import하면 모든 컴포넌트에 기본 스타일이 적용됩니다.

```ts
import '@woon-ui/react/css'
```

필요한 컴포넌트만 선택적으로 가져올 수도 있습니다.

```ts
import '@woon-ui/dialog/css/dialog'   // Dialog 프리미티브
import '@woon-ui/dialog/css/alert'    // alert() 프리셋
import '@woon-ui/dialog/css/confirm'  // confirm() 프리셋
import '@woon-ui/toast/css'    // Toast
import '@woon-ui/popover/css'  // Popover
import '@woon-ui/tooltip/css'  // Tooltip
```

### 스타일 커스텀

특정 값만 바꾸고 싶다면 `[data-woon-*]` 선택자로 override하세요.

```css
[data-woon-dialog-content] {
  background: #1a1a1a;
  border-radius: 12px;
}
```

### 완전 교체

기본 스타일을 완전히 걷어내고 처음부터 직접 스타일링하려면, `@woon-ui/react/css` import를 제거하고 각 컴포넌트 문서 하단의 CSS 소스를 복사해서 수정하세요.

<Callout type="tip">
  Tailwind를 사용한다면 `[data-woon-*]` 선택자에 `@apply`를 쓰거나, TSX 컴포넌트를 복사해서 클래스를 직접 적용할 수 있습니다.
</Callout>

## TypeScript

`@woon-ui/react`는 TypeScript로 작성되어 타입 정의가 내장되어 있습니다.
별도의 `@types` 패키지가 필요하지 않습니다.

React 19의 새로운 기능을 사용하므로, `tsconfig.json`에서 React 타입이 19 이상인지 확인하세요.

```json
{
  "compilerOptions": {
    "jsx": "react-jsx"
  }
}
```
