## 패키지 설치

```bash
npm install @woon/core
```

```bash
pnpm add @woon/core
```

```bash
yarn add @woon/core
```

<Callout type="note">
  `@woon/core`는 `react >= 19.0.0`과 `react-dom >= 19.0.0`을 peer dependency로 요구합니다.
</Callout>

## Provider 설정

앱 루트에 `WoonProvider`를 추가하고, 사용할 기능을 플러그인으로 주입합니다.

```tsx
import { WoonProvider } from '@woon/core'
import { dialogPlugin } from '@woon/core/dialog'
import { toastPlugin } from '@woon/core/toast'

export default function App() {
  return (
    <WoonProvider
      plugins={[
        dialogPlugin(),
        toastPlugin({ position: 'bottom-right' }),
      ]}
    >
      <YourApp />
    </WoonProvider>
  )
}
```

필요한 플러그인만 추가하면 됩니다. 예를 들어 Dialog만 사용한다면:

```tsx
<WoonProvider plugins={[dialogPlugin()]}>
  <YourApp />
</WoonProvider>
```

### WoonProvider Props

| Prop | Type | Default | 설명 |
|------|------|---------|------|
| `plugins` | `WoonPlugin[]` | `[]` | 사용할 플러그인 배열 |
| `baseZIndex` | `number` | `200` | 오버레이 z-index 시작값 |

<Callout type="tip">
  `toast()`, `alert()`, `confirm()` 같은 명령형 API는 WoonProvider 없이도 동작합니다.
  Provider는 렌더 위치와 전역 설정을 연결하는 역할입니다.
</Callout>

## CSS 가져오기

앱 진입점에 CSS를 한 번 import하면 모든 컴포넌트에 기본 스타일이 적용됩니다.

```ts
import '@woon/core/css'
```

필요한 컴포넌트만 선택적으로 가져올 수도 있습니다.

```ts
import '@woon/core/css/dialog'
import '@woon/core/css/toast'
import '@woon/core/css/popover'
import '@woon/core/css/tooltip'
```

### 스타일 커스텀

특정 값만 바꾸고 싶다면 `[data-woon-*]` 선택자로 override하세요.

```css
[data-woon-dialog-content] {
  background: #1a1a1a;
  border-radius: 12px;
}
```

### 완전 교체

기본 스타일을 완전히 걷어내고 처음부터 직접 스타일링하려면, `@woon/core/css` import를 제거하고 각 컴포넌트 문서 하단의 CSS 소스를 복사해서 수정하세요.

<Callout type="tip">
  Tailwind를 사용한다면 `[data-woon-*]` 선택자에 `@apply`를 쓰거나, TSX 컴포넌트를 복사해서 클래스를 직접 적용할 수 있습니다.
</Callout>

## TypeScript

`@woon/core`는 TypeScript로 작성되어 타입 정의가 내장되어 있습니다.
별도의 `@types` 패키지가 필요하지 않습니다.

React 19의 새로운 기능을 사용하므로, `tsconfig.json`에서 React 타입이 19 이상인지 확인하세요.

```json
{
  "compilerOptions": {
    "jsx": "react-jsx"
  }
}
```
