## WoonProvider

`WoonProvider`는 overlay 기능의 렌더 위치와 전역 설정을 연결하는 컴포넌트입니다.
앱 진입점에 한 번만 등록하면 됩니다.

```tsx
export function Providers({ children }: { children: React.ReactNode }) {
  return (
    <WoonProvider plugins={[dialogPlugin(), toastPlugin()]}>
      {children}
    </WoonProvider>
  )
}
```

`plugins` prop에 사용할 플러그인 배열을 전달합니다. 순서는 상관없습니다.

<Callout type="note">
  `alert()`, `confirm()`, `toast()`는 `WoonProvider` 없이도 호출할 수 있습니다. 단, Provider가 없으면 화면에 렌더되지 않습니다.
</Callout>

## 플러그인

각 플러그인의 상세 옵션은 해당 컴포넌트 문서를 참고하세요.

| 플러그인 | 기능 | 문서 |
|---------|------|------|
| `dialogPlugin` | `useDialog()`, `alert()`, `confirm()` | [Dialog](/docs/components/dialog) |
| `toastPlugin` | `toast()` | [Toast](/docs/components/toast) |
