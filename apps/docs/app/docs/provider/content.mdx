```ts
import { WoonProvider } from '@woon/core'
import { dialogPlugin } from '@woon/core/dialog'
import { toastPlugin } from '@woon/core/toast'
```

## WoonProvider

`WoonProvider`는 overlay 기능의 렌더 위치와 전역 설정을 연결하는 컴포넌트입니다.
앱 진입점에 한 번만 등록하면 됩니다.

```tsx
export function Providers({ children }: { children: React.ReactNode }) {
  return (
    <WoonProvider plugins={[dialogPlugin(), toastPlugin()]}>
      {children}
    </WoonProvider>
  )
}
```

`plugins` prop에 사용할 플러그인 배열을 전달합니다. 순서는 상관없습니다.

<Callout type="note">
  `alert()`, `confirm()`, `toast()`는 `WoonProvider` 없이도 호출할 수 있습니다. Provider는 다이얼로그·토스트가 실제로 렌더될 위치를 잡아주는 역할입니다. Provider 없이 호출하면 렌더가 되지 않습니다.
</Callout>

## 플러그인

### dialogPlugin

`useDialog()`, `alert()`, `confirm()`을 사용하려면 `dialogPlugin`을 등록해야 합니다.

```tsx
dialogPlugin({
  zIndex: 200,              // 기본 z-index (쌓일수록 1씩 증가)
  scrollTarget: '#main',    // scrollLock 기본 대상 (미지정 시 document.body)
  render: {                 // alert() / confirm() 커스텀 컴포넌트
    alert: MyAlert,
    confirm: MyConfirm,
  },
})
```

| 옵션 | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `zIndex` | `number` | `200` | 다이얼로그 z-index 시작값. 중첩될수록 1씩 증가합니다. |
| `scrollTarget` | `string \| Element` | `document.body` | `scrollLock`의 기본 잠금 대상. `height: 100dvh` 레이아웃에서 실제 스크롤 컨테이너를 지정할 때 사용합니다. `dialog.open()` 옵션으로 호출별 오버라이드 가능합니다. |
| `render.alert` | `ComponentType<AlertRenderContext>` | 내장 UI | `alert()` 기본 렌더 컴포넌트를 교체합니다. |
| `render.confirm` | `ComponentType<ConfirmRenderContext>` | 내장 UI | `confirm()` 기본 렌더 컴포넌트를 교체합니다. |

### toastPlugin

`toast()`를 사용하려면 `toastPlugin`을 등록해야 합니다.

```tsx
toastPlugin({
  position: 'bottom-right',  // 토스트 표시 위치
  maxVisible: 3,              // 동시에 표시할 최대 개수
  zIndex: 9000,               // 컨테이너 z-index
})
```

| 옵션 | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `position` | `'top-left' \| 'top-center' \| 'top-right' \| 'bottom-left' \| 'bottom-center' \| 'bottom-right'` | `'bottom-right'` | 토스트가 나타나는 위치입니다. |
| `maxVisible` | `number` | `3` | 화면에 동시에 표시할 최대 토스트 수. 초과 시 가장 오래된 토스트부터 제거됩니다. |
| `zIndex` | `number` | `9000` | 토스트 컨테이너의 z-index입니다. |
