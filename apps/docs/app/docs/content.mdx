## Headless UI

headless 컴포넌트 + 기본 CSS를 제공하며, 스타일은 완전히 오버라이드 가능합니다.
React 19 기반으로 설계되어 `ref`를 일반 prop처럼 사용할 수 있습니다.

## 핵심 원칙

- **UX 강제** — ESC 닫기, 포커스 트랩, 스크롤 잠금이 기본 동작입니다. 접근성은 선택이 아닌 기본값입니다.
- **헤드리스** — 기능은 완성, 스타일은 자유. 기본 CSS를 제공하되, 전부 덮어쓸 수 있습니다.
- **명령형 API** — `dialog.open()`, `toast()`, `confirm()` 등 선언형과 명령형을 함께 지원합니다.
- **플러그인 아키텍처** — 필요한 기능만 플러그인으로 주입. 트리쉐이킹에 유리합니다.

## 빠른 시작

```bash
npm install @woon/core
```

```tsx
import { WoonProvider } from '@woon/core'
import { dialogPlugin } from '@woon/core/dialog'
import { toastPlugin } from '@woon/core/toast'
import '@woon/core/css'

export default function App() {
  return (
    <WoonProvider
      plugins={[
        dialogPlugin(),
        toastPlugin({ position: 'bottom-right' }),
      ]}
    >
      <YourApp />
    </WoonProvider>
  )
}
```
