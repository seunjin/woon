## 스냅 포인트

`snapPoints` prop에 뷰포트 높이 비율 배열을 전달해 여러 스냅 위치를 정의할 수 있습니다. 값은 `0`~`1` 사이로, 예를 들어 `0.4`는 뷰포트의 40% 높이를 의미합니다.

`defaultSnap`으로 초기 스냅 인덱스를 지정합니다. 생략 시 마지막 인덱스(가장 높은 스냅)로 열립니다.

```tsx
<BottomSheet.Root snapPoints={[0.4, 0.9]} defaultSnap={0}>
  {/* 처음 열릴 때 40% 높이, 위로 드래그하면 90%로 확장 */}
</BottomSheet.Root>
```

## 스크롤 가능한 콘텐츠

콘텐츠가 바텀 시트 높이를 초과할 때는 스크롤 영역이 필요합니다. `data-woon-bottom-sheet-body` 속성을 가진 `div`를 사용하면 해당 영역이 `flex: 1`로 늘어나며 내부 스크롤이 활성화됩니다.

```tsx
<BottomSheet.Content>
  <BottomSheet.Handle />
  <div style={{ padding: '0 1.5rem 0.75rem' }}>
    <BottomSheet.Title>제목</BottomSheet.Title>
  </div>
  <div data-woon-bottom-sheet-body>
    {items.map((item) => (
      <div key={item}>{item}</div>
    ))}
  </div>
</BottomSheet.Content>
```

### 한계 — 복잡한 내부 레이아웃

`data-woon-bottom-sheet-body`는 **단일 스크롤 영역** 하나만 있는 단순한 구조 전용 편의 도구입니다. 내부에서 `height: 100%`를 상속받을 수 없기 때문에, 다음과 같은 복잡한 레이아웃에서는 사용하지 마세요.

- 스크롤 목록 위에 서브 헤더가 있는 경우
- 스크롤 영역 아래에 고정 액션 버튼이 있는 경우
- 스크롤 영역이 여러 개인 경우

이런 경우 `data-woon-bottom-sheet-body`를 쓰지 않고, `BottomSheet.Content`가 이미 `flex-direction: column`인 점을 활용해 직접 레이아웃을 구성하세요.

```tsx
<BottomSheet.Content>
  <BottomSheet.Handle />

  {/* 고정 헤더 */}
  <div style={{ padding: '0 1.5rem 0.75rem', flexShrink: 0 }}>
    <BottomSheet.Title>제목</BottomSheet.Title>
  </div>

  {/* 스크롤 영역 — flex: 1로 남은 공간 차지 */}
  <div style={{ flex: 1, overflowY: 'auto', overscrollBehavior: 'contain' }}>
    {items.map((item) => (
      <div key={item}>{item}</div>
    ))}
  </div>

  {/* 고정 하단 버튼 */}
  <div style={{ padding: '1rem 1.5rem', flexShrink: 0, borderTop: '1px solid #e5e7eb' }}>
    <button type="button" style={{ width: '100%' }}>확인</button>
  </div>
</BottomSheet.Content>
```

## 제어 모드

`open`과 `onOpenChange`를 전달하면 외부에서 열림 상태를 직접 제어할 수 있습니다. `BottomSheet.Trigger` 없이도 버튼이나 다른 이벤트로 열 수 있습니다.

```tsx
const [open, setOpen] = useState(false)

<button onClick={() => setOpen(true)}>열기</button>

<BottomSheet.Root open={open} onOpenChange={setOpen}>
  <BottomSheet.Content>
    <BottomSheet.Handle />
    <button onClick={() => setOpen(false)}>닫기</button>
  </BottomSheet.Content>
</BottomSheet.Root>
```

## API

### BottomSheet.Root

<PropsTable rows={bottomSheetRootRows} />

### BottomSheet.Content

바텀 시트 패널 본체입니다. Portal을 통해 `document.body`에 렌더됩니다. `children` 외 표준 HTML `div` 속성을 모두 지원합니다.

<PropsTable rows={bottomSheetContentRows} />

### BottomSheet.Handle

드래그 핸들 바입니다. `aria-hidden="true"`가 자동으로 적용됩니다. 이 요소를 잡고 드래그해서 스냅 포인트 간 이동하거나 닫을 수 있습니다. 표준 HTML `div` 속성을 지원합니다 (단, 내부적으로 사용하는 포인터 이벤트 핸들러는 override할 수 없습니다).

## 스타일

`@woon-ui/bottom-sheet/css`로 기본 스타일이 적용됩니다. 특정 값만 바꾸려면 선택자로 override하세요.

```css
[data-woon-bottom-sheet-content] {
  border-radius: 16px 16px 0 0;
  background: #1e1e2e;
  color: #fff;
}
[data-woon-bottom-sheet-handle]::before {
  background: #444;
}
```

완전히 교체하려면 `@woon-ui/bottom-sheet/css` import를 제거하고 아래 CSS를 복사해서 수정하세요.

## 접근성

- `role="dialog"`, `aria-modal="true"` 자동 적용
- 열릴 때 첫 번째 포커서블 요소로 포커스 이동
- 닫힐 때 이전 포커스 요소로 복귀
- ESC로 닫기 가능
- 배경(overlay) 클릭으로 닫기
